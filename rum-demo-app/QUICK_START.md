# Quick Start - RUM Demo App

Get the demo app running in 3 steps!

## Step 1: Install

```bash
cd rum-demo-app
npm install
```

## Step 2: Start

```bash
npm run dev
```

App opens at `http://localhost:3001`

## Step 3: Generate Events

1. Click buttons in the app
2. Navigate between pages
3. Generate errors
4. View events in dashboard at `http://localhost:3000`

## Prerequisites

- Backend running on `http://localhost:8080`
- Dashboard running on `http://localhost:3000`

See `SETUP_GUIDE.md` in root directory for complete setup.
